# פרומפטים לעיבוד תביעות

## 1. הערכת תביעה ראשונית

**שימוש:** הערכה ראשונית מהירה של תביעה חדשה
**מודל מומלץ:** Claude / ChatGPT
**משתנים:** `{טופס_תביעה}`, `{פוליסה}`

### פרומפט:
```
אתה בודק תביעות ביטוח בריאות מנוסה.

בצע הערכה ראשונית של התביעה מול הפוליסה:

1. **חילוץ פרטים:** מספר תביעה, תאריך, סוג אירוע, סכום
2. **בדיקת כיסוי:** האם האירוע מכוסה בפוליסה?
3. **בדיקת אכשרה:** האם תקופת האכשרה הסתיימה?
4. **חישוב סכום:** סכום מבוקש פחות השתתפות עצמית
5. **מסמכים חסרים:** האם כל המסמכים הנדרשים צורפו?
6. **דגלים:** נקודות שדורשות תשומת לב מיוחדת

פלט בטבלה:
| פרמטר | ערך | סטטוס |
|-------|-----|-------|

תביעה:
{טופס_תביעה}

פוליסה:
{פוליסה}
```

## 2. טיוטת מכתב אישור תביעה

**שימוש:** הכנת מכתב אישור מקצועי ללקוח
**מודל מומלץ:** ChatGPT (כתיבה טבעית בעברית)
**משתנים:** `{שם_מבוטח}`, `{מספר_תביעה}`, `{סכום}`, `{פרטי_אירוע}`

### פרומפט:
```
כתוב מכתב אישור תביעת ביטוח בריאות מקצועי ואמפתי.

פרטים:
- שם מבוטח: {שם_מבוטח}
- מספר תביעה: {מספר_תביעה}
- סכום מאושר: {סכום}
- פרטי האירוע: {פרטי_אירוע}

המכתב צריך לכלול:
1. פנייה אישית
2. אישור קבלת התביעה
3. פירוט הסכום המאושר
4. פירוט חישוב (סכום מקורי - השתתפות עצמית = סכום לתשלום)
5. אופן התשלום ולוח זמנים
6. פרטי קשר לשאלות

הטון: מקצועי, חם, ברור
```

## 3. טיוטת מכתב דחייה

**שימוש:** הכנת מכתב דחייה מנומק וברור
**מודל מומלץ:** ChatGPT
**משתנים:** `{שם_מבוטח}`, `{מספר_תביעה}`, `{סיבת_דחייה}`, `{סעיף_רלוונטי}`

### פרומפט:
```
כתוב מכתב דחיית תביעת ביטוח. המכתב חייב להיות:
- מקצועי ומכבד
- ברור בסיבת הדחייה
- מפנה לסעיף הרלוונטי בפוליסה
- מציע אפשרויות (ערעור, פנייה לממונה)

פרטים:
- שם מבוטח: {שם_מבוטח}
- מספר תביעה: {מספר_תביעה}
- סיבת דחייה: {סיבת_דחייה}
- סעיף בפוליסה: {סעיף_רלוונטי}

מבנה המכתב:
1. פנייה אישית
2. אזכור התביעה והאירוע
3. החלטה: התביעה נדחתה
4. נימוק מפורט עם הפניה לסעיף
5. הסבר בשפה פשוטה למה הכיסוי לא חל
6. זכות ערעור: כיצד ולמי לפנות
7. פרטי קשר ונכונות לסייע

⚠️ חובה לכלול: זכות ערעור ופנייה לממונה על פניות הציבור
```

## 4. סיכום תיק תביעה

**שימוש:** הכנת סיכום תיק לדיון צוות או העברה
**מודל מומלץ:** Claude
**משתנים:** `{כל_מסמכי_התיק}`

### פרומפט:
```
סכם את תיק התביעה הבא בצורה מובנית לצורך דיון צוות.

כלול:
1. **ציר זמן:** אירועים מרכזיים בסדר כרונולוגי
2. **סיכום רפואי:** אבחנות וטיפולים
3. **סיכום כספי:** סכומים מבוקשים vs מאושרים
4. **סטטוס נוכחי:** היכן עומד הטיפול
5. **סוגיות פתוחות:** מה צריך להחליט/לברר
6. **המלצה:** הצעה לצעד הבא

{כל_מסמכי_התיק}
```

## 5. בדיקת שלמות תביעה

**שימוש:** וידוא שכל המסמכים הנדרשים צורפו
**מודל מומלץ:** ChatGPT / Claude
**משתנים:** `{סוג_תביעה}`, `{מסמכים_שצורפו}`

### פרומפט:
```
בדוק שלמות תביעת ביטוח בריאות.

סוג התביעה: {סוג_תביעה}

מסמכים שצורפו:
{מסמכים_שצורפו}

רשימת מסמכים נדרשים לפי סוג:
- אשפוז: טופס תביעה, מכתב שחרור, דוח ניתוח, קבלות, אישור אשפוז
- ביקור מומחה: טופס תביעה, הפניה, סיכום ביקור, קבלה
- בדיקות: טופס תביעה, הפניה, תוצאות, קבלה
- תרופות: טופס תביעה, מרשם, קבלה מבית מרקחת

הצג:
| מסמך | נדרש | צורף | הערה |
|------|------|------|------|

ציין אילו מסמכים חסרים ומה צריך לבקש מהמבוטח.
```
