# פרומפטים לחיתום

## 1. ניתוח שאלון בריאות

**שימוש:** ניתוח תשובות שאלון בריאות של מבקש ביטוח
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
אתה חתם ביטוח חיים ובריאות מנוסה בחברת ביטוח ישראלית.

נתח את שאלון הבריאות הבא וזהה:
1. **גורמי סיכון:** רשימה ממוספרת עם דירוג חומרה (נמוך/בינוני/גבוה)
2. **מצבים רפואיים מדווחים:** רשימה עם הערכת משמעות
3. **פערים/חסרים:** מידע שלא צוין אבל כדאי לברר
4. **תרופות:** ניתוח משמעות התרופות (מעידות על מצבים רפואיים)
5. **היסטוריה משפחתית:** גורמי סיכון תורשתיים

סיווג סיכון מומלץ:
- סטנדרט (S): ללא גורמי סיכון משמעותיים
- מוגבר (R): גורמי סיכון שדורשים תוספת פרמיה
- דחייה (D): סיכון גבוה מדי
- מידע נוסף (I): צריך לברר לפני החלטה

{שאלון_בריאות}
```

## 2. המלצת חיתום

**שימוש:** הכנת המלצת חיתום מנומקת
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
הכן המלצת חיתום מפורטת למבקש הביטוח.

פרטי מבקש:
- גיל: {גיל}
- מגדר: {מגדר}
- סוג ביטוח מבוקש: {סוג_ביטוח}
- סכום ביטוח: {סכום}

ממצאים רפואיים:
{ממצאים}

מבנה ההמלצה:
1. **סיכום גורמי סיכון:** טבלה עם גורם + חומרה
2. **סיווג מומלץ:** סטנדרט / מוגבר / דחייה
3. **תוספת פרמיה:** אם מוגבר - כמה אחוז ולמה
4. **חריגות מוצעות:** אם יש - מה להחריג ולכמה זמן
5. **תנאים מיוחדים:** בדיקות נוספות, תקופת המתנה
6. **נימוק:** הסבר מפורט להחלטה

⚠️ הערה: זוהי המלצה בלבד. ההחלטה הסופית בידי חתם מוסמך.
```

## 3. יצירת שאלות הבהרה למבקש

**שימוש:** כשצריך מידע נוסף מהמבקש לפני החלטת חיתום
**מודל מומלץ:** ChatGPT

### פרומפט:
```
בהתבסס על שאלון הבריאות הבא, הכן רשימת שאלות הבהרה למבקש.

לכל שאלה ציין:
- השאלה (בשפה ברורה ומכבדת)
- למה צריך את המידע (הערה פנימית)
- מסמכים שיש לצרף

פורמט המכתב:
"שלום {שם},
תודה על פנייתך לביטוח. לצורך השלמת הבדיקה, נודה לקבל מידע נוסף:
1. ...
2. ...
אנא צרף/י את המסמכים הבאים: ..."

{שאלון_בריאות}
{ממצאים_חריגים}
```

## 4. השוואת סיכון מול טבלאות

**שימוש:** הערכת סיכון מול קריטריוני חיתום
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
העריך את הסיכון של המבקש לפי הקריטריונים הבאים.

פרופיל המבקש:
{פרופיל}

טבלת קריטריונים:
| פרמטר | סטנדרט | מוגבר | דחייה |
|-------|--------|-------|-------|
| BMI | 18.5-30 | 30-35 | 35+ |
| לחץ דם סיסטולי | <140 | 140-160 | 160+ |
| HbA1c | <6.5% | 6.5-8% | 8%+ |
| עישון | לא | <10/יום | 10+/יום |
| כולסטרול | <200 | 200-250 | 250+ |

הצג טבלת תוצאות:
| פרמטר | ערך המבקש | קטגוריה | הערות |
```

```

פרטי מבקש: {פרטים}
ממצאים: {ממצאים}
בדיקות נוספות: {בדיקות}

מבנה הדוח:
═══ דוח חיתום ═══
מספר בקשה: 
תאריך: 
חתם: 

1. פרטי המבקש (גיל, מגדר, עיסוק)
2. סוג ביטוח מבוקש וסכום
3. סיכום רפואי
4. גורמי סיכון שזוהו
5. בדיקות שבוצעו/התבקשו
6. החלטת חיתום
7. תנאים מיוחדים (אם יש)
8. נימוק

חתימה: ___________
```
