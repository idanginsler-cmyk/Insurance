# פרומפטים לרגולציה וציות

## 1. בדיקת ציות של מסמך שיווקי

**שימוש:** בדיקה שחומר שיווקי עומד בדרישות רגולטוריות
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
אתה יועץ רגולציה בתחום הביטוח בישראל.

בדוק את החומר השיווקי הבא מול עקרונות הפיקוח על שוק הביטוח:

1. **דיוק:** האם כל הטענות מדויקות ולא מטעות?
2. **שקיפות:** האם יש גילוי מלא של תנאים, חריגות, ועלויות?
3. **אותיות קטנות:** האם יש מידע חשוב שמוסתר?
4. **הטעיה:** האם יש ניסוח שעלול להטעות צרכן סביר?
5. **השוואות:** אם יש השוואה למתחרים - האם היא הוגנת?
6. **הבטחות:** האם יש הבטחות שלא ניתן לקיים?

לכל ממצא:
- ציטוט מהטקסט
- הבעיה
- חומרה: נמוכה / בינונית / גבוהה
- הצעת תיקון

{חומר_שיווקי}
```

## 2. בדיקת עמידה בחובת גילוי

**שימוש:** וידוא שמסמכי לקוח כוללים את כל הגילויים הנדרשים
**מודל מומלץ:** Claude

### פרומפט:
```
בדוק האם מסמך ההצעה/פוליסה הבא כולל את כל חובות הגילוי:

רשימת בדיקה:
- [ ] גילוי מלא של כל החריגות
- [ ] ציון ברור של תקופות אכשרה
- [ ] פירוט השתתפות עצמית לכל כיסוי
- [ ] הסבר זכות ביטול (14 יום)
- [ ] הסבר תהליך הגשת תביעה
- [ ] ציון חובת הגילוי של המבוטח
- [ ] פירוט השלכות אי-גילוי
- [ ] פרטי קשר לפניות ציבור
- [ ] ציון זכות פנייה לממונה על שוק ההון

סמן עבור כל סעיף: ✅ קיים / ❌ חסר / ⚠️ חלקי

{מסמך}
```

## 3. ניתוח השפעת שינוי רגולטורי

**שימוש:** הערכת השפעה של חוזר/תקנה חדשים על הפעילות
**מודל מומלץ:** Claude

### פרומפט:
```
נתח את השפעת השינוי הרגולטורי הבא על פעילות חברת ביטוח חיים ובריאות.

השינוי:
{תיאור_השינוי}

נתח:
1. **תחומי השפעה:** אילו מחלקות/תהליכים מושפעים
2. **שינויים נדרשים:** מה צריך לשנות (מוצרים, תהליכים, מערכות, הדרכות)
3. **לוח זמנים:** עד מתי צריך ליישם
4. **עלויות משוערות:** השקעה נדרשת
5. **סיכונים:** מה קורה אם לא מיישמים בזמן
6. **הזדמנויות:** האם יש יתרון תחרותי ביישום מוקדם

הצג כטבלת השפעה:
| תחום | שינוי נדרש | עדיפות | מחלקה אחראית |
```

## 4. הכנת דוח ציות תקופתי

**שימוש:** סיוע בהכנת דוח ציות לפיקוח
**מודל מומלץ:** Claude

### פרומפט:
```
סייע בהכנת מתווה לדוח ציות רבעוני/שנתי.

תחומי ביקורת:
{תחומים}

מבנה הדוח:
1. **סיכום מנהלים:** ממצאים עיקריים (10 שורות)
2. **ממצאים לפי תחום:**
   - תחום הממצא
   - תיאור
   - חומרה (קריטי/גבוה/בינוני/נמוך)
   - סטטוס (חדש/בטיפול/נסגר)
   - צעדי תיקון
   - מועד יעד
3. **מגמות:** שיפור/הרעה ביחס לתקופה קודמת
4. **המלצות:** פעולות מתקנות מומלצות
5. **נספחים:** ראיות ומסמכים תומכים

⚠️ הערה: דוח זה הוא טיוטה לעיבוד ע"י צוות ציות מוסמך.
```
