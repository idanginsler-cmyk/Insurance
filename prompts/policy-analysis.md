# פרומפטים לניתוח פוליסות

## 1. סיכום פוליסה מקיף

**שימוש:** קבלת סיכום מהיר של פוליסת ביטוח ארוכה
**מודל מומלץ:** Claude (חלון הקשר גדול) / ChatGPT
**משתנים:** `{טקסט_הפוליסה}`

### פרומפט:
```
אתה מומחה ביטוח בריאות ישראלי עם 20 שנות ניסיון.

נתח את פוליסת הביטוח הבאה וצור סיכום מובנה:

1. **פרטי פוליסה:** מספר, סוג, תאריכי תוקף
2. **כיסויים עיקריים:** רשימה עם סכומי ביטוח מקסימליים
3. **חריגות מהותיות:** מה לא מכוסה
4. **תקופות אכשרה:** לכל כיסוי
5. **השתתפות עצמית:** סכומים לכל סוג שירות
6. **תנאים מיוחדים:** הגבלות, תנאים לחידוש, ביטול
7. **נקודות לתשומת לב:** סעיפים חריגים או יוצאי דופן

הצג כטבלה מסודרת עם כותרות ברורות.

הפוליסה:
{טקסט_הפוליסה}
```

### טיפים:
- עבור פוליסות ארוכות מאוד (100+ עמודים), השתמשו ב-Claude
- אם הפלט ארוך מדי, בקשו: "סכם ב-10 נקודות בלבד"

## 2. השוואת פוליסות

**שימוש:** השוואה בין שתי פוליסות או יותר
**מודל מומלץ:** Claude
**משתנים:** `{פוליסה_א}`, `{פוליסה_ב}`

### פרומפט:
```
אתה מומחה ביטוח בריאות. השווה בין שתי הפוליסות הבאות.

צור טבלת השוואה מפורטת:

| כיסוי | פוליסה א | פוליסה ב | הערות |
|-------|---------|---------|-------|

כלול: כיסויים, סכומי ביטוח, חריגות, השתתפות עצמית, תקופות אכשרה.

בסוף, תן:
1. סיכום: 3 יתרונות עיקריים של כל פוליסה
2. המלצה מנומקת: איזו פוליסה מקיפה יותר

פוליסה א:
{פוליסה_א}

פוליסה ב:
{פוליסה_ב}
```

## 3. זיהוי חריגות וסעיפים בעייתיים

**שימוש:** בדיקת פוליסה לסעיפים שעלולים להפתיע את המבוטח
**מודל מומלץ:** Claude / ChatGPT
**משתנים:** `{טקסט_הפוליסה}`

### פרומפט:
```
אתה עורך דין המתמחה בדיני ביטוח ישראליים.

סקור את הפוליסה הבאה וזהה:
1. **חריגות מפתיעות** - כיסויים שנראים כלולים אבל בפועל מוחרגים
2. **סעיפים מגבילים** - תנאים שמצמצמים משמעותית את הכיסוי
3. **ניסוח עמום** - סעיפים שניתנים לפרשנות כפולה
4. **תקופות אכשרה ארוכות** - מעל הסטנדרט בשוק
5. **תנאי ביטול** - סעיפים שמאפשרים ביטול חד-צדדי

לכל ממצא ציין:
- סעיף מספר/מיקום
- ציטוט מהטקסט
- למה זה בעייתי
- מה מקובל בשוק לשם השוואה

{טקסט_הפוליסה}
```

## 4. פישוט שפה משפטית

**שימוש:** תרגום סעיפי פוליסה מ"שפה משפטית" לעברית פשוטה
**מודל מומלץ:** ChatGPT / Claude
**משתנים:** `{סעיף}`

### פרומפט:
```
הפוך את סעיף הביטוח הבא לשפה פשוטה וברורה שלקוח ללא רקע משפטי יבין.

כללים:
- השתמש במשפטים קצרים
- הימנע ממונחים משפטיים - תרגם אותם
- תן דוגמה מעשית להמחשה
- שמור על הדיוק המשפטי

הסעיף המקורי:
{סעיף}

פורמט התשובה:
**מקור:** [הסעיף המקורי]
**בפשטות:** [ההסבר הפשוט]
**דוגמה:** [תרחיש מעשי שממחיש]
```

**משתנים:** `{טקסט_הפוליסה}`

### פרומפט:
```
חלץ את כל הנתונים המובנים מפוליסת הביטוח הבאה לפורמט JSON.

מבנה JSON נדרש:
{
  "policy_number": "",
  "policy_type": "",
  "start_date": "",
  "end_date": "",
  "insured": {
    "name": "",
    "id": "",
    "birth_date": ""
  },
  "premium_monthly": 0,
  "coverages": [
    {
      "name": "",
      "max_amount": 0,
      "deductible": 0,
      "waiting_period_months": 0,
      "notes": ""
    }
  ],
  "exclusions": [""],
  "special_conditions": [""]
}

אם שדה לא מופיע במסמך, רשום null.

{טקסט_הפוליסה}
```
