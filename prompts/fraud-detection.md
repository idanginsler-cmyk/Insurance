# פרומפטים לזיהוי הונאות

## 1. ניתוח תביעה לדגלים אדומים

**שימוש:** סריקת תביעה חדשה לאיתור סימני הונאה
**מודל מומלץ:** Claude / ChatGPT
**משתנים:** `{פרטי_התביעה}`

### פרומפט:
```
אתה חוקר הונאות ביטוח מנוסה. נתח את התביעה הבאה ובדוק דגלים אדומים.

בדוק את הפרמטרים הבאים:
1. **עיתוי:** האם התביעה הוגשה זמן קצר אחרי תחילת הפוליסה?
2. **תדירות:** האם יש תביעות קודמות רבות מאותו מבוטח?
3. **סכום:** האם הסכום חריג ביחס לסוג האירוע?
4. **תיאור:** האם יש אי-התאמות בתיאור האירוע?
5. **מסמכים:** האם המסמכים התומכים מלאים ועקביים?
6. **דפוסים:** האם יש דמיון לתבניות הונאה ידועות?

לכל דגל אדום שזוהה:
- תאר את הממצא
- דרג חומרה: נמוך / בינוני / גבוה
- הסבר למה זה חשוד

בסוף, תן ציון סיכון כולל: 1-10 (1=נקי, 10=חשוד מאוד)
והמלצה: לאשר / לבדוק / להעביר לחקירה

{פרטי_התביעה}
```

## 2. זיהוי אי-התאמות בתיאורים

**שימוש:** השוואת תיאורים שונים של אותו אירוע
**מודל מומלץ:** Claude
**משתנים:** `{תיאור_1}`, `{תיאור_2}`

### פרומפט:
```
השווה בין שני התיאורים הבאים של אותו אירוע ביטוחי.

זהה:
1. **סתירות:** עובדות שסותרות זו את זו
2. **פערים:** מידע שמופיע בתיאור אחד ולא בשני
3. **שינויי גרסה:** פרטים שהשתנו בין התיאורים
4. **דיוקים חשודים:** פרטים שנראים "מושלמים מדי" או מתורגלים

תיאור 1 (של המבוטח):
{תיאור_1}

תיאור 2 (מהמסמכים/עדים):
{תיאור_2}

הצג ממצאים בטבלה:
| ממצא | תיאור 1 | תיאור 2 | חומרה |
```

3. **דפוס חוזר:** תביעות דומות בתדירות גבוהה
4. **אסקלציה:** תביעות שהולכות וגדלות בסכום

תביעה חדשה:
{תביעה_חדשה}

תביעות קודמות:
{תביעות_קודמות}

## 4. יצירת שאלות חקירה

**שימוש:** הכנת שאלות לחקירת תביעה חשודה
**מודל מומלץ:** ChatGPT
**משתנים:** `{פרטי_התביעה}`, `{דגלים_אדומים}`

### פרומפט:
```
אתה חוקר הונאות ביטוח. על בסיס התביעה והדגלים האדומים שזוהו,
הכן רשימת שאלות חקירה למבוטח.

דרישות:
- 10-15 שאלות
- התחל משאלות כלליות וסיים בספציפיות
- כלול שאלות שבודקות עקביות בגרסה
- כלול שאלות על פרטים שקשה "להמציא"
- ציין לכל שאלה: מה אתה מנסה לבדוק

תביעה:
{פרטי_התביעה}

דגלים אדומים:
{דגלים_אדומים}
```

## 5. סיכום ממצאי חקירה

**שימוש:** כתיבת דוח סיכום חקירת הונאה
**מודל מומלץ:** Claude / ChatGPT
**משתנים:** `{ממצאים}`

### פרומפט:
```
כתוב דוח סיכום חקירת הונאה מקצועי בהתבסס על הממצאים הבאים.

מבנה הדוח:
1. **פרטי התביעה:** מספר, תאריך, סכום, סוג
2. **סיבת פתיחת חקירה:** מה עורר חשד
3. **ממצאים עיקריים:** 3-5 ממצאים מרכזיים
4. **ראיות:** מסמכים/עדויות שתומכים בממצאים
5. **מסקנה:** האם יש ביסוס לחשד הונאה
6. **המלצה:** אישור / דחייה / המשך חקירה
7. **צעדים נדרשים:** מה צריך לעשות בהמשך

כתוב בשפה מקצועית ועובדתית, ללא שיפוטיות.

ממצאים:
{ממצאים}
```
