# פרומפטים לשירות לקוחות

## 1. תשובה לפניית לקוח

**שימוש:** הכנת תשובה מקצועית ואמפתית לפניית לקוח
**מודל מומלץ:** ChatGPT (כתיבה טבעית בעברית)

### פרומפט:
```
כתוב תשובה מקצועית ואמפתית לפניית הלקוח הבאה.

כללים:
- טון: חם, מקצועי, מכבד
- אורך: 100-200 מילים
- פתח בהתייחסות אישית לפנייה
- תן תשובה ברורה ומדויקת
- סיים בנכונות לעזור

פניית הלקוח:
{פניה}

מידע רלוונטי מהפוליסה:
{מידע_פוליסה}
```

## 2. הסבר פוליסה בשפה פשוטה

**שימוש:** הסבר סעיף או כיסוי ללקוח בשפה מובנת
**מודל מומלץ:** ChatGPT

### פרומפט:
```
הסבר ללקוח בשפה פשוטה וברורה את הכיסוי הבא מפוליסת הביטוח שלו.

כללים:
- אל תשתמש במונחים משפטיים
- תן דוגמה מעשית
- הסבר מה מכוסה ומה לא
- ציין סכומים והשתתפות עצמית

הסעיף מהפוליסה:
{סעיף}

פורמט:
**מה זה אומר בפשטות:**
[הסבר]

**דוגמה מעשית:**
[תרחיש]

**חשוב לדעת:**
[נקודות מפתח]
```

## 3. טיפול בתלונה

**שימוש:** הכנת תשובה מקצועית לתלונת לקוח
**מודל מומלץ:** ChatGPT

### פרומפט:
```
כתוב תשובה לתלונת הלקוח הבאה.

עקרונות:
1. הכר בתחושת הלקוח (אמפתיה)
2. הסבר מה קרה ולמה
3. אם יש טעות שלנו - הודה והצע פתרון
4. אם אין טעות - הסבר בעדינות ובבהירות
5. הצע צעד הבא קונקרטי
6. ציין את זכות הפנייה לממונה על פניות הציבור

התלונה:
{תלונה}

עובדות הרקע:
{עובדות}
```

## 4. הודעת חידוש פוליסה

**שימוש:** הכנת הודעת חידוש שנתית מותאמת אישית
**מודל מומלץ:** ChatGPT

### פרומפט:
```
כתוב הודעת חידוש פוליסה אישית ומזמינה.

פרטי הלקוח:
- שם: {שם}
- סוג פוליסה: {סוג}
- ותק: {שנים} שנים
- שינוי בפרמיה: {שינוי}

המכתב צריך לכלול:
1. פנייה אישית וחמה
2. תודה על הנאמנות (אם ותיק)
3. סיכום קצר של הכיסויים
4. שינויים לשנה הקרובה (אם יש)
5. עדכון פרמיה (עם הסבר אם יש עלייה)
6. הזמנה ליצירת קשר לשאלות
```

## 5. FAQ - תשובות לשאלות נפוצות

**שימוש:** יצירת בנק תשובות לשאלות נפוצות
**מודל מומלץ:** ChatGPT / Claude

### פרומפט:
```
צור תשובות לשאלות הנפוצות הבאות בנושא ביטוח בריאות.

לכל שאלה:
- תשובה קצרה (2-3 משפטים)
- תשובה מורחבת (אם הלקוח רוצה יותר פרטים)

שאלות:
1. איך מגישים תביעה?
2. מה זה תקופת אכשרה?
3. מה זה השתתפות עצמית?
4. האם ביקור אצל מומחה דורש הפניה?
5. מה עושים במקרה חירום בחו"ל?
6. איך מבטלים פוליסה?
7. מה קורה אם לא שילמתי פרמיה?
8. האם מצב רפואי קודם מכוסה?
9. איך מוסיפים בן משפחה לפוליסה?
10. מה ההבדל בין ביטוח בריאות פרטי לסל הבריאות?

כתוב בשפה פשוטה שכל לקוח יבין.
```
