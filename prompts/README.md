# תבניות פרומפט לביטוח

תיקייה זו מכילה תבניות פרומפט מוכנות לשימוש בתחומי ביטוח חיים ובריאות.

## איך להשתמש

כל קובץ מכיל מספר תבניות פרומפט. לכל תבנית:
- **שימוש:** תיאור מתי להשתמש
- **מודל מומלץ:** איזה כלי AI הכי מתאים
- **משתנים:** מה צריך להחליף בפרומפט (מסומן ב-`{סוגריים}`)
- **הפרומפט:** הטקסט לשליחה ל-AI
- **דוגמה:** קלט ופלט צפוי
- **טיפים:** איך להתאים לצרכים שלכם

## כללי זהב לפרומפטים בביטוח

1. **תמיד ציינו הקשר:** "אתה מומחה ביטוח בריאות ישראלי..."
2. **בקשו פורמט מובנה:** טבלאות, רשימות, JSON
3. **הגדירו מגבלות:** "אם אין מספיק מידע, ציין זאת"
4. **דרשו ביקורתיות:** "ציין גם סיכונים/בעיות אפשריות"
5. **אל תזינו מידע אמיתי:** השתמשו במידע אנונימי/סינתטי בלבד
