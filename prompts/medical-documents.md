# פרומפטים למסמכים רפואיים

## 1. סיכום מסמך רפואי

**שימוש:** סיכום מהיר של מכתב שחרור או סיכום רפואי
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
אתה רופא בודק תביעות בחברת ביטוח בריאות ישראלית.

סכם את המסמך הרפואי הבא וחלץ:
1. **אבחנות:** רשימה עם קודי ICD-10 אם רלוונטי
2. **טיפולים שבוצעו:** ניתוחים, פרוצדורות, טיפולים
3. **תרופות:** שם, מינון, תדירות
4. **בדיקות ותוצאות:** בדיקות שבוצעו עם ממצאים
5. **מצבים קודמים:** מחלות רקע ומצבים pre-existing
6. **המלצות:** המשך טיפול, מעקבים, הגבלות
7. **ימי אי-כושר:** אם צוין

הצג בטבלה מסודרת.

{מסמך_רפואי}
```

## 2. מיפוי ממצאים רפואיים לכיסוי ביטוחי

**שימוש:** בדיקה אילו טיפולים מכוסים בפוליסה
**מודל מומלץ:** Claude (חלון הקשר גדול לשני מסמכים)

### פרומפט:
```
בהתבסס על הסיכום הרפואי והפוליסה, נתח:

| טיפול/פרוצדורה | עלות | מכוסה? | סעיף בפוליסה | השתתפות עצמית | לתשלום |
|---------------|------|--------|-------------|--------------|--------|

בסוף, ציין:
- סכום כולל לתשלום למבוטח
- טיפולים שלא מכוסים ולמה
- תקופות אכשרה רלוונטיות

סיכום רפואי:
{מסמך_רפואי}

פוליסה:
{פוליסה}
```

## 3. זיהוי מצבים רפואיים קודמים

**שימוש:** בדיקה האם יש pre-existing conditions
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
אתה חתם רפואי. נתח את המסמך הבא וזהה מצבים רפואיים קודמים (pre-existing conditions).

לכל מצב שזוהה:
1. **המצב:** שם + קוד ICD
2. **תאריך אבחון ראשוני:** מתי אובחן לראשונה
3. **קשר לאירוע הנוכחי:** האם יש קשר סיבתי
4. **השפעה על כיסוי:** האם זה משפיע על הכיסוי הביטוחי
5. **מסקנה:** האם יש בסיס לחריגת pre-existing

⚠️ הערה: מצב רפואי קודם רלוונטי רק אם אובחן לפני תחילת הפוליסה
ובתוך תקופת ההחרגה הקבועה בפוליסה.

{מסמך_רפואי}
```

## 4. תרגום מסמך רפואי

**שימוש:** תרגום מסמך רפואי עם שימור מונחים מקצועיים
**מודל מומלץ:** Claude / ChatGPT

### פרומפט:
```
תרגם את המסמך הרפואי הבא מ{שפת_מקור} לעברית.

כללים:
1. שמור על מונחים רפואיים בשפת המקור בסוגריים
   דוגמה: "אי ספיקת לב (Heart Failure)"
2. שמור על קודי ICD ושמות תרופות באנגלית
3. המר יחידות מידה לסטנדרט ישראלי אם צריך
4. סמן מונחים שתרגומם לא ודאי ב-[?]

{מסמך_רפואי}
```

## 5. הכנת סיכום רפואי לחתם

**שימוש:** הכנת סיכום תמציתי לחתם מתוך מסמכים רפואיים מרובים
**מודל מומלץ:** Claude

### פרומפט:
```
הכן סיכום רפואי תמציתי לחתם ביטוח חיים/בריאות.

המסמכים:
{מסמכים_רפואיים}

פורמט נדרש:
**גיל ומגדר:** 
**מחלות רקע:** (רשימה ממוספרת)
**ניתוחים קודמים:** (רשימה עם שנים)
**תרופות קבועות:** (שם + מינון)
**אשפוזים ב-5 שנים אחרונות:** (תאריך + סיבה)
**גורמי סיכון:** (עישון, BMI, לחץ דם, סוכרת, משפחתי)
**מגבלות תפקודיות:** (אם יש)
**הערכת סיכון כוללת:** נמוך / בינוני / מוגבר / גבוה
**המלצה:** סטנדרט / תוספת פרמיה / דחייה / מידע נוסף
```
