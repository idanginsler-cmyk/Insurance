# 🚀 מדריך הקמת Databricks Community Edition

## שלב 1: פתיחת חשבון (5 דקות)

1. כנס ל: https://www.databricks.com/try-databricks
2. בחר **"Get started for free"**
3. בחר **"Community Edition"** (חשוב! לא Azure/AWS)
4. מלא פרטים עם אימייל **אישי** (לא של העבודה)
5. אשר אימייל

## שלב 2: יצירת Cluster (3 דקות)

1. בתפריט שמאל: **Compute**
2. לחץ **Create compute**
3. שם: `my-cluster`
4. הכל ברירת מחדל
5. **Create cluster**
6. חכה ~3 דקות עד שהמעגל יהפוך ירוק

## שלב 3: העלאת CSV

1. בתפריט: **Data → Create Table**
2. לחץ **Upload File**
3. גרור את `claims_2025.csv`
4. שים לב לנתיב (בד"כ `/FileStore/tables/claims_2025.csv`)
5. לחץ **Create Table in Notebook** (או Cancel - נעשה בעצמנו)

## שלב 4: יצירת Notebook

1. בתפריט: **Workspace → Users → [your email]**
2. לחץ ימני → **Create → Notebook**
3. שם: `fraud_analysis`
4. שפה: **Python**
5. Cluster: בחר את `my-cluster`

## שלב 5: העתקת הקוד

1. פתח את הקובץ `fraud_analysis_databricks.py`
2. העתק את כל התוכן
3. הדבק ב-Notebook (הוא יכיר את מבנה התאים אוטומטית)
4. לחץ **Run All** (כפתור למעלה)

## שלב 6: צפייה בתוצאות

תראה:
- 📊 סטטיסטיקות
- 🔴 מבוטחים חשודים
- 🏥 ניתוח נותני שירות
- 👥 אשכולות משפחתיים
- 📅 ניתוח חודשי
- 🎯 ציוני סיכון
- גרפים אינטראקטיביים אוטומטיים!

## שלב 7: יצירת Dashboard

1. בכל תוצאה - לחץ על **"+"** ובחר **"Add to dashboard"**
2. או: **Workspace → Create → Dashboard**
3. בחר מה להציג
4. שמור

## שלב 8: שיתוף הקישור

1. למעלה מימין: **Share**
2. העתק את הקישור
3. פתח בכל מחשב (כולל בעבודה) - דורש התחברות לחשבון שלך
