# 🔍 מנתח הונאות - Java

כלי עצמאי לניתוח תביעות ביטוח בריאות. **ללא תלויות חיצוניות** - רק Java.

## מה הוא עושה
קורא קובץ CSV של תביעות → מזהה דפוסי הונאה → מייצר דוח HTML

### דפוסים שנבדקים:
1. **תדירות חריגה** - מבוטחים עם הרבה תביעות
2. **ריכוז נותן שירות** - ריכוז גבוה מדי
3. **סכום אחיד** - נותן שירות עם אותו סכום חוזר
4. **סכומים עגולים** - כפולות של 500₪
5. **אשכולות משפחתיים** - משפחות שכולן אצל אותו נ.ש.

## מה צריך
- **JDK** (Java Development Kit) - לקימפול
- **JRE** - להרצה (יש לך בעבודה)

## איך לקמפל (בבית או במחשב עם JDK)

```bash
javac FraudAnalyzer.java
```

זה ייצור קובץ `FraudAnalyzer.class`

## איך להריץ (בכל מחשב עם Java)

שים באותה תיקייה:
- `FraudAnalyzer.class`
- קובץ ה-CSV שלך (בשם `claims.csv`)

הרץ:
```bash
java FraudAnalyzer
```

או עם שמות מותאמים:
```bash
java FraudAnalyzer myfile.csv output.html
```

## תוצאה
יווצר קובץ `fraud-report.html` - פתח אותו בדפדפן ותראה את הדוח המלא.

## פורמט CSV נדרש
עמודות שהכלי מצפה להן (באנגלית, בשורה הראשונה):
- `claim_id` - מספר תביעה
- `date` - תאריך (YYYY-MM-DD)
- `insured_name` - שם מבוטח
- `insured_id` - ת.ז.
- `age` - גיל
- `city` - עיר
- `employer` - מעסיק
- `policy_type` - סוג פוליסה
- `policy_start` - תחילת פוליסה (YYYY-MM-DD)
- `policy_num` - מספר פוליסה
- `coverage` - כיסוי
- `provider` - נותן שירות
- `amount_claimed` - סכום נתבע
- `amount_paid` - סכום שולם
