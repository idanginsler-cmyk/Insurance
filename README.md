# תכנית הדרכה - AI בביטוח חיים ובריאות

תוכנית הכשרה מקיפה לשילוב כלי בינה מלאכותית בעבודת ביטוח חיים ובריאות.

## מטרות

1. **הקניית ידע ופרקטיקה** - יצירת כלי עבודה מהירים באמצעות AI
2. **הכשרת מובילי ידע** - העובדים ילמדו להטמיע AI בתהליכי העבודה היומיומיים
3. **חיסכון בזמן** - אוטומציה של פעולות חוזרות: סיכום מסמכים, ניתוח פוליסות, עיבוד תביעות

## מבנה התוכנית - 5 מפגשים

| # | מפגש | נושאים מרכזיים |
|---|-------|----------------|
| 1 | **מבוא והיכרות** | סקירת עולם ה-AI, דוגמאות מתעשיית הביטוח |
| 2 | **סקירת כלים קיימים** | ChatGPT, Claude, Gemini, Copilot, Perplexity + התאמה לביטוח |
| 3 | **עיבוד מסמכים מהיר** | סיכום פוליסות, ניתוח מסמכים רפואיים, תרגום, שאלות על מסמכים |
| 4 | **בניית סוכנים חכמים** | אוטומציות, סוכני AI לחיתום/הונאות/תביעות, דשבורדים |
| 5 | **הטמעת חשיבת AI** | שינוי תרבותי, מדיניות ארגונית, רגולציה, תוכנית 90 יום |

## מבנה הריפוזיטורי

```
docs/           - מדריך הדרכה מלא, קטלוג כלים, מילון מונחים
sessions/       - חומרי מפגש: מתווה, תרגילים, מסמכי דוגמה
prompts/        - תבניות פרומפט מוכנות לשימוש בתחומי ביטוח
demos/          - סקריפטים Python להדגמה חיה
```

## התחלה מהירה

```bash
# 1. שכפול הריפוזיטורי
git clone <repo-url>
cd Insurance

# 2. יצירת סביבה וירטואלית
python -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate   # Windows

# 3. התקנת תלויות
pip install -r requirements.txt

# 4. הגדרת מפתחות API
cp .env.example .env
# ערוך את .env עם המפתחות שלך

# 5. הרצת דמו
python demos/demo_1_policy_analyzer.py
```

ראה [SETUP.md](SETUP.md) להוראות מפורטות.
