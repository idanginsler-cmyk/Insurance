## נקודות דיבור מרכזיות

### מה זה AI Agent?
- **צ'אטבוט:** עונה על שאלה אחת בכל פעם
- **AI Agent:** מקבל מטרה → מתכנן שלבים → מבצע פעולות → מחזיר תוצאה
- **אנלוגיה:** צ'אטבוט = מילון | סוכן AI = עובד חדש שמקבל משימה ויודע לבצע אותה

### דוגמה: סוכן עיבוד תביעות
```
קלט: טופס תביעה חדש
  ↓
שלב 1: חילוץ פרטים מהטופס (שם, מספר פוליסה, סוג אירוע)
  ↓
שלב 2: שליפת פוליסה מהמערכת (API)
  ↓
שלב 3: בדיקה - האם האירוע מכוסה בפוליסה?
  ↓
שלב 4: חישוב סכום - בהתאם לכיסוי ולהשתתפות עצמית
  ↓
שלב 5: בדיקת דגלים אדומים (הונאה?)
  ↓
שלב 6: הכנת טיוטת החלטה + מכתב
  ↓
פלט: דוח מובנה לבודק תביעות אנושי
```

### כלים לבניית סוכנים

#### ללא קוד (No-Code) - מומלץ לעובדים עסקיים
- **Power Automate + AI Builder:** אוטומציות בתוך Microsoft 365
  - דוגמה: מייל נכנס עם מסמך → AI מסכם → מעביר לצוות הרלוונטי
  - דוגמה: טופס תביעה → חילוץ נתונים → הזנה למערכת
- **Zapier AI:** חיבור בין אפליקציות עם שכבת AI
  - דוגמה: לקוח שולח מייל → AI מסווג (תביעה/שאלה/תלונה) → מנתב

#### עם קוד - לצוותי IT ופיתוח
- **LangChain:** ספריית Python לשרשור פעולות AI
  - חיבור LLM + מאגרי מידע + APIs + כלים
- **CrewAI:** צוות סוכנים שעובדים יחד
  - דוגמה: חוקר + מנתח + כותב = צוות שמטפל בתביעה מורכבת

### שימושים בביטוח

#### 1. סוכן זיהוי הונאות
- מנתח תביעות חדשות מול דפוסים חשודים
- בודק: תביעות כפולות, אי-התאמות בתיאורים, עיתוי חשוד
- מדרג סיכון הונאה (נמוך/בינוני/גבוה)
- מייצר דוח חקירה למחלקת הונאות

#### 2. סוכן חיתום
- מקבל שאלון בריאות ונתוני מבקש
- מנתח גורמי סיכון
- מסווג סיכון (סטנדרט/מוגבר/סירוב)
- מייצר המלצת חיתום עם נימוקים

#### 3. סוכן שירות לקוחות חכם
- עונה על שאלות נפוצות על פוליסות
- מנתב פניות מורכבות לגורם המתאים
- מייצר טיוטות תשובה למוקדנים

#### 4. דשבורד AI
- ניתוח מגמות תביעות בזמן אמת
- זיהוי אנומליות בנתונים
- חיזוי עלויות תביעות

### אינטגרציה עם מערכות קיימות
- **SharePoint:** גישה למסמכי פוליסות ותביעות
- **מערכת CRM:** שליפת נתוני לקוחות
- **מערכת תביעות:** עדכון סטטוסים
- **דוא"ל (Outlook):** שליחת תשובות אוטומטיות
