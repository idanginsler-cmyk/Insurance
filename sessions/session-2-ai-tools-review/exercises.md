# תרגילים - מפגש 2: סקירת כלים

## תרגיל 1: השוואת כלים - אותה משימה ב-3 כלים (קבוצתי)
**רמה:** מתחילים | **זמן:** 20 דקות

### הוראות:
1. כל קבוצה תקבל את אותו טקסט (סעיף מפוליסת ביטוח)
2. שלחו את הטקסט ל-ChatGPT, Claude ו-Gemini עם אותו פרומפט:

```
סכם את הסעיף הבא מפוליסת ביטוח בריאות ב-5 נקודות.
ציין: מה מכוסה, מה לא מכוסה, ומה דורש אישור מראש.

[הדביקו את הטקסט כאן]
```

3. השוו את התוצאות:
   - איזה סיכום הכי מדויק?
   - איזה הכי קריא?
   - האם מישהו פספס משהו חשוב?

## תרגיל 2: Prompt Engineering - שיפור הדרגתי (אישי)
**רמה:** מתחילים-בינוני | **זמן:** 15 דקות

### הוראות:
התחילו עם פרומפט בסיסי ושפרו אותו בכל שלב:

**שלב 1 - פרומפט בסיסי:**
```
מה זה ביטוח חיים?
```

**שלב 2 - הוספת הקשר:**
```
אני סוכן ביטוח חדש. הסבר לי מה זה ביטוח חיים ריסק לעומת ביטוח חיים מעורב.
```

**שלב 3 - בקשת פורמט ספציפי:**
```
אני סוכן ביטוח חדש. הכן לי טבלת השוואה בין ביטוח חיים ריסק לביטוח חיים מעורב.
הטבלה צריכה לכלול: סוג הכיסוי, משך הביטוח, מרכיב חיסכון, עלות יחסית, למי מתאים.
```

**שלב 4 - הוספת תפקיד ומגבלות:**
```
אתה מומחה ביטוח חיים ישראלי עם 20 שנות ניסיון.
הכן לי טבלת השוואה בין ביטוח חיים ריסק לביטוח חיים מעורב.
הטבלה צריכה לכלול: סוג הכיסוי, משך הביטוח, מרכיב חיסכון, עלות יחסית, למי מתאים.
הוסף שורה עם "המלצה" - למי כדאי כל אחד.
כתוב בעברית, בשפה פשוטה שלקוח יבין.
```

### שאלות לדיון:
- איך השתפרה התוצאה בכל שלב?
- מה הכי השפיע על איכות התשובה?

## תרגיל 4: יצירת מצגת עם AI (אישי)
**רמה:** מתחילים | **זמן:** 15 דקות

### הוראות:
1. היכנסו ל-Gamma.app
2. כתבו בתיבת הטקסט:
```
מצגת על יתרונות ביטוח בריאות פרטי בישראל - 5 שקופיות
```
3. בחרו עיצוב ולחצו "Generate"
4. בדקו את התוצאה: מה טוב? מה צריך לתקן?
5. נסו לשפר שקופית אחת ידנית

### למתקדמים:
נסו ליצור את אותה מצגת ב-Beautiful.ai או Tome והשוו.
