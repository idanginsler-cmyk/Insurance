# מפגש 3 - עיבוד מסמכים מהיר עם AI

## פרטי המפגש
- **משך:** 3 שעות
- **קהל יעד:** כלל המשתתפים
- **דרישות מקדימות:** מפגשים 1-2, גישה ל-Copilot/ChatGPT/Claude

## מטרות למידה
1. לעבד מסמכי ביטוח במהירות: סיכום, חילוץ מידע, תרגום
2. לנתח מסמכים רפואיים ולמפות ממצאים לכיסוי ביטוחי
3. להשוות פוליסות ולזהות הבדלי כיסוי
4. לשאול שאלות חכמות על מסמכים

## נקודות דיבור מרכזיות

### עקרונות עיבוד מסמכים
- **סיכום:** קבלת תמצית של מסמך ארוך ב-5 נקודות
- **חילוץ (Extraction):** שליפת נתונים ספציפיים - סכומים, תאריכים, שמות
- **סיווג:** קטגוריזציה אוטומטית של מסמכים (תביעה, פוליסה, מכתב)
- **תרגום:** תרגום עם שימור מונחים מקצועיים
- **שאלות ותשובות:** שאילת שאלות ספציפיות על מסמך

### סיכום פוליסת ביטוח
**פרומפט מומלץ:**
```
נתח את פוליסת הביטוח הבאה וחלץ:
1. סוג הביטוח ומספר פוליסה
2. כיסויים עיקריים וסכומי ביטוח
3. חריגות מהותיות
4. תקופת אכשרה
5. השתתפות עצמית
6. תנאים מיוחדים

הצג כטבלה מסודרת.
```

### ניתוח מסמכים רפואיים
**שימושים:**
- חילוץ אבחנות וקודי ICD
- זיהוי מצבים רפואיים קודמים (pre-existing conditions)
- סיכום היסטוריה רפואית
- מיפוי ממצאים לכיסוי בפוליסה

**⚠️ אזהרה חשובה:** לעולם אל תזינו מסמכים רפואיים **אמיתיים** של מבוטחים לכלי AI חיצוניים! בהדרכה נשתמש במסמכים **סינתטיים** בלבד.

### השוואת פוליסות
**פרומפט מומלץ:**
```
השווה בין שתי הפוליסות הבאות. הצג טבלת השוואה עם:
- כיסויים קיימים בכל פוליסה
- הבדלים בסכומי ביטוח
- הבדלים בחריגות
- הבדלים בהשתתפות עצמית
- המלצה: איזו פוליסה מקיפה יותר ולמה

פוליסה א: [...]
פוליסה ב: [...]
```

### תרגום מקצועי
- AI יכול לתרגם מסמכים רפואיים מאנגלית/ערבית לעברית
- חשוב לבקש: "שמור על מונחים רפואיים מקוריים בסוגריים"
- תמיד לבדוק תרגום מונחים מקצועיים ע"י גורם מוסמך
