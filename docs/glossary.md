# מילון מונחים: AI וביטוח

## מונחי AI ולמידת מכונה

| מונח בעברית | English Term | הגדרה |
|------------|-------------|-------|
| בינה מלאכותית | Artificial Intelligence (AI) | תחום במדעי המחשב שעוסק ביצירת מערכות שמבצעות משימות הדורשות "חשיבה" |
| למידת מכונה | Machine Learning (ML) | תת-תחום של AI - מחשב שלומד מדוגמאות ומשתפר בביצועים |
| למידה עמוקה | Deep Learning | למידת מכונה באמצעות רשתות נוירונים עם שכבות רבות |
| מודל שפה גדול | Large Language Model (LLM) | מודל AI שאומן על מיליארדי מסמכים ויודע להבין ולייצר טקסט |
| רשת נוירונים | Neural Network | מבנה מתמטי בהשראת המוח האנושי, הבסיס ללמידה עמוקה |
| פרומפט | Prompt | ההוראה או השאלה שמזינים למודל AI |
| הנדסת פרומפטים | Prompt Engineering | האמנות של כתיבת פרומפטים אפקטיביים לקבלת תוצאות טובות |
| הזיה | Hallucination | כשמודל AI "ממציא" מידע שנשמע משכנע אבל שגוי |
| כוונון עדין | Fine-tuning | התאמת מודל קיים לתחום ספציפי באמצעות נתונים ייעודיים |
| RAG | Retrieval Augmented Generation | טכניקה שמחברת חיפוש במאגר מידע עם יצירת טקסט AI |
| טוקן | Token | יחידת טקסט בסיסית (בערך מילה אחת) - יחידת המדידה של LLMs |
| חלון הקשר | Context Window | כמות הטקסט שמודל יכול לעבד בבת אחת (נמדד בטוקנים) |
| הטבעה | Embedding | ייצוג מתמטי של טקסט כווקטור מספרים - לחיפוש ודמיון |
| סוכן AI | AI Agent | מערכת AI שמקבלת מטרה, מתכננת שלבים, ומבצעת פעולות אוטונומיות |
| אוטומציה | Automation | הפיכת תהליך ידני לתהליך אוטומטי |
| זיהוי תווים אופטי | OCR (Optical Character Recognition) | טכנולוגיה שממירה טקסט בתמונה/סריקה לטקסט דיגיטלי |
| עיבוד שפה טבעית | NLP (Natural Language Processing) | תחום AI שעוסק בהבנה ועיבוד של שפה אנושית |
| צ'אטבוט | Chatbot | תוכנה שמנהלת שיחה עם משתמשים בשפה טבעית |
| ממשק תכנות | API (Application Programming Interface) | דרך לתקשר עם שירות תוכנה באופן אוטומטי |
| טמפרטורה | Temperature | פרמטר ששולט ברמת היצירתיות של מודל AI (0=מדויק, 1=יצירתי) |
| למידה מדוגמאות | Few-shot Learning | כשנותנים למודל מספר דוגמאות בפרומפט כדי שילמד את הפורמט |
| למידה ללא דוגמאות | Zero-shot Learning | כשמודל מבצע משימה ללא דוגמאות - רק מהוראה |
| מודל בסיס | Foundation Model | מודל AI גדול ורב-תכליתי שאומן על נתונים רחבים |
| אימון | Training | תהליך הלמידה של מודל AI מנתונים |
| הסקה | Inference | הרצת מודל מאומן על קלט חדש - "לשאול שאלה" |

## מונחי ביטוח

| מונח בעברית | English Term | הגדרה |
|------------|-------------|-------|
| חיתום | Underwriting | תהליך הערכת סיכונים והחלטה על קבלה/דחייה של בקשת ביטוח |
| פוליסה | Policy | חוזה הביטוח - מסמך המגדיר את הכיסויים, התנאים והחריגות |
| פרמיה | Premium | התשלום התקופתי (חודשי/שנתי) עבור הביטוח |
| תביעה | Claim | פנייה של מבוטח לקבלת תגמולים בעקבות אירוע ביטוחי |
| כיסוי | Coverage | ההגנה הביטוחית - מה הפוליסה כוללת |
| חריגה | Exclusion | מקרה או מצב שלא מכוסה בפוליסה |
| השתתפות עצמית | Deductible | הסכום שהמבוטח משלם מכיסו בכל תביעה |
| מוטב | Beneficiary | האדם שמקבל את תגמולי הביטוח |
| סוכן ביטוח | Insurance Agent | מי שמתווך בין חברת הביטוח ללקוח |
| אקטואר | Actuary | מומחה לחישוב סיכונים סטטיסטיים ותמחור ביטוח |
| סיכון | Risk | ההסתברות וההשפעה של אירוע ביטוחי |
| שיפוי | Indemnity | פיצוי כספי - החזרת המבוטח למצב שלפני הנזק |
| ביטוח חיים | Life Insurance | ביטוח שמשלם למוטבים במקרה מוות של המבוטח |
| ביטוח בריאות | Health Insurance | ביטוח שמכסה הוצאות רפואיות |
| ביטוח סיעודי | Long-term Care Insurance | ביטוח שמכסה מצב סיעודי - צורך בעזרה יומיומית |
| מצב רפואי קודם | Pre-existing Condition | מחלה שאובחנה לפני תחילת הביטוח |
| תקופת אכשרה | Waiting Period | תקופה מתחילת הביטוח שבה אין כיסוי לאירועים מסוימים |
| ביטוח משנה | Reinsurance | ביטוח של חברת ביטוח אצל חברה אחרת - פיזור סיכונים |
| הונאת ביטוח | Insurance Fraud | הגשת תביעה כוזבת או מניפולציה לקבלת תגמולים שלא מגיעים |
| הפיקוח על הביטוח | Insurance Commissioner | הרגולטור הישראלי שמפקח על שוק הביטוח |

|------------|-------------|-------|
| חיתום אוטומטי | Automated Underwriting | שימוש ב-AI להערכת סיכונים וקבלת החלטות חיתום |
| זיהוי הונאות | Fraud Detection | שימוש ב-AI לאיתור תביעות חשודות ודפוסי הונאה |
| עיבוד תביעות חכם | Smart Claims Processing | אוטומציה של תהליך עיבוד תביעות באמצעות AI |
| ניתוח סיכונים | Risk Analytics | ניתוח נתונים מתקדם להערכת סיכונים ביטוחיים |
| שירות לקוחות חכם | AI Customer Service | צ'אטבוטים וכלי AI לשיפור שירות הלקוחות |
| אינשורטק | InsurTech | חברות טכנולוגיה שמשנות את עולם הביטוח |
| תמחור דינמי | Dynamic Pricing | חישוב פרמיות מותאם אישית בהתבסס על נתוני AI |
| ניתוח מסמכים רפואיים | Medical Document Analysis | שימוש ב-AI לחילוץ מידע ממסמכים רפואיים |
| ציות רגולטורי | Regulatory Compliance | עמידה בדרישות הרגולטור באמצעות כלי AI |

| הערכת נזק אוטומטית | Automated Damage Assessment | שימוש ב-AI להערכת היקף נזק מתמונות/מסמכים |
